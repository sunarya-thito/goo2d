# Contributing to Goo2D

Thank you for your interest in contributing to Goo2D! We welcome contributions from everyone.

## How to Contribute

1. **Fork the Repository**: Create your own fork of the Goo2D repository.
2. **Create a Branch**: Work on your changes in a new branch.
3. **Write Code**: Implement your feature or fix.
4. **Test Your Changes**: Ensure everything works as expected.
5. **Submit a Pull Request**: Submit your PR with a clear description of the changes.

## Code Style

Please follow the [official Dart style guide](https://dart.dev/guides/language/effective-dart/style). Run `flutter format .` before submitting your PR.

## Community

Join our discord or open an issue for discussion!
